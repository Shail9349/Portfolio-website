## How to Use

### Adding a New Application
1. Go to "Add Application" section
2. Fill in the job title and organization (required)
3. Select application date and pay any fee (if applicable)
4. Add exam date and admit card date (if known)
5. Select current status from the dropdown
6. Add any additional notes
7. Click "Add Application"

### Tracking Progress
Update application status as you progress:
- **Applied**: Application submitted
- **Admit Card Downloaded**: Admit card available
- **Exam Given**: Exam completed
- **Result Declared**: Result announced
- **Selected**: Successfully selected
- **Rejected**: Application rejected

### Getting Reminders
- The app automatically generates reminders for:
  - Admit card availability (7 days before)
  - Upcoming exams (14 days before)
- Notifications appear in the bell icon (top-right)
- Daily reminders show urgent deadlines

### Data Management
- **Export**: Download all your data for backup
- **Import**: Restore data from previous backup
